\doxysubsection{Detailed Description}
Utility class for parsing SVG files and generating corresponding shapes. 

The \mbox{\hyperlink{classSVGParser}{SVGParser}} class provides static methods for parsing SVG files and creating shape objects based on the content of the file.

The list of supported shapes currently includes\+:
\begin{DoxyItemize}
\item Rectangles 
\end{DoxyItemize}

This function reads an SVG file line by line, extracts SVG elements, and creates corresponding shape objects. It supports a finite list of SVG elements which are defined in the {\ttfamily ELEMENT\+\_\+\+NAMES} array. The parsed shapes are returned in a vector.

The parsing process involves\+:
\begin{DoxyEnumerate}
\item Opening the SVG file.
\item Reading each line and extracting SVG elements by finding their {\ttfamily START\+\_\+\+MARKER} and {\ttfamily END\+\_\+\+MARKER}.
\item Instantiating shape objects, passing the SVG elements.
\item Adding the created shapes to a vector.
\item Returning the vector of shapes.
\end{DoxyEnumerate}
