\doxysubsection{Detailed Description}
Utility class for parsing SVG files and generating corresponding shapes. 

The \mbox{\hyperlink{classSVGParser}{SVGParser}} class provides static methods for parsing SVG files and creating shape objects based on the content of the files. It supports parsing of specific SVG elements like rectangles.
