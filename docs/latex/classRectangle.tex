\doxysubsection{Detailed Description}
Represents a rectangular shape, implementing the \mbox{\hyperlink{classShape}{Shape}} interface. 

The \mbox{\hyperlink{classRectangle}{Rectangle}} class provides functionality to work with rectangular shapes in SVG elements and G-\/code instructions.
